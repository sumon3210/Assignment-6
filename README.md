# 🚀 DigiTools - Premium Digital Tools Platform

## 📌 Project Overview
DigiTools is a modern and responsive web application that provides a collection of premium digital tools like AI writing, design templates, automation tools, and more. Users can explore products, add them to cart, and simulate a checkout experience.

---

## ✨ Description
This project is built with a clean UI/UX design and focuses on delivering a smooth user experience. It includes a dynamic cart system, interactive product cards, and a fully responsive layout for mobile and desktop devices.

---

## 🛠️ Technologies Used
- ⚛️ React JS
- 🎨 Tailwind CSS
- 🧩 Lucide React Icons
- 💡 JavaScript (ES6)
- 📦 Vite

---

## 🔥 Key Features

### 🛒 1. Smart Cart System
- Add products to cart
- Remove items easily
- Real-time total price calculation

### 📱 2. Fully Responsive Design
- Mobile, Tablet & Desktop friendly
- Smooth navigation menu
- Sidebar cart for better UX

### ⚡ 3. Interactive UI Experience
- Animated buttons & hover effects
- Dynamic product listing
- Instant "Added to Cart" feedback
